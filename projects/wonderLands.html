Wonderlands – Airbnb Replica

Over View

Wonderlands is a full-stack web application inspired by Airbnb, where users can explore listings, view them on an interactive map, add ratings & reviews, and manage their own properties securely. The project focuses on backend architecture, authentication, CRUD operations, and real-world deployment.

Features

  • User authentication & authorization using Passport.js.
  • Create, edit, delete listings (only by authorized owners).
  • Add ratings (out of 5) and reviews with user permissions.
  • Interactive maps using Leaflet.js with location markers.
  • Image uploads & storage via Cloudinary.
  • Server-side form validation using Joi.
  • Fully responsive UI with Bootstrap.
  • Robust backend using Express middleware, controllers, and REST APIs.

Tech Stack

  • Node.js
  • Express.js
  • EJS (Templating)
  • MongoDB & Mongoose
  • Bootstrap
  • HTML, CSS, JavaScript
  • Leaflet.js (Maps)
  • Cloudinary (Image Storage)
  • Render (Deployment)

What I Learnt

  • Building full-stack applications with authentication and authorization.
  • Creating and structuring REST APIs, routes, controllers, and middleware.
  • Working with databases, schema design, and Mongoose relationships.
  • Handling file uploads, integrating Maps and Cloudinary.
  • Implementing session management and flash messaging.
  • Deploying full-stack apps.
  • Improving debugging, error handling, and backend architecture.
